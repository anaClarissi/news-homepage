### 📚 What I learned

During this project, I strengthened my understanding of:

* Combining **CSS Grid and Flexbox** for complex layouts
* Building a **responsive navigation menu**
* Structuring styles using **CSS variables**
* Using the `<picture>` element for responsive images

Example of a grid layout I implemented:

```css
main {
  display: grid;
  grid-template-columns: 2fr 1fr;
  grid-template-areas: 
    "principal news"
    "cards cards";
}
```
